# nx

NixOS and Home Manager dotfiles for `zen`, with shared system/home modules, host-specific monitor and disk layout files, agenix scaffolding, and a standalone Home Manager build.

## Architecture

```text
~/nx
├── flake.nix
├── lib/default.nix
├── hosts/
│   └── zen/
│       ├── default.nix        # NixOS entrypoint
│       ├── hardware.nix       # generated by nixos-generate-config
│       ├── disko.nix          # declarative disk layout
│       ├── monitors.nix       # host-specific monitor/workspace layout
│       └── home.nix           # Home Manager entrypoint
├── modules/
│   ├── nixos/                # shared NixOS modules
│   └── home/                 # shared Home Manager modules
├── secrets/
│   ├── secrets.nix
│   └── *.age
└── wallpapers/
```

## Common Commands

```bash
nix fmt .
nix flake check

home-manager build --flake ~/nx#wyattgill@zen
home-manager switch --flake ~/nx#wyattgill@zen

sudo nixos-rebuild build --flake ~/nx#zen
sudo nixos-rebuild switch --flake ~/nx#zen
```

## Quick Start For A New Host

1. Copy `hosts/zen/` to `hosts/<new-host>/`.
2. Replace `hardware.nix`, `disko.nix`, and `monitors.nix` with the new machine’s values.
3. Add a new host definition in `flake.nix` with its `hostName`, `username`, `email`, `homeDirectory`, and SSH public keys.
4. Point a `nixosConfigurations.<new-host>` entry at `./hosts/<new-host>/default.nix`.
5. Add a matching `homeConfigurations."user@host"` entry for `./hosts/<new-host>/home.nix`.
6. Run `nix fmt .`, `nix flake check`, `home-manager build --flake`, and `sudo nixos-rebuild build --flake` before switching.

## Secrets Management

The repo ships with valid encrypted placeholder files so the agenix workflow is ready immediately.

Public key policy lives in `secrets/secrets.nix`. To replace a placeholder:

```bash
cd ~/nx/secrets

printf 'your-tailscale-auth-key\n' | RULES=./secrets.nix nix run github:ryantm/agenix -- -e tailscale-authkey.age
printf 'your-password-hash\n' | RULES=./secrets.nix nix run github:ryantm/agenix -- -e user-password.age
```

Secret application is intentionally opt-in. Enable it in `hosts/zen/default.nix` only after the placeholders are replaced:

```nix
nx.nixos.security = {
  manageTailscaleAuthKey = true;
  manageUserPassword = true;
};
```

## Notes

- `modules/home/desktop/common.nix` is the shared source of truth for terminal, launcher, file manager, workspace numbers, and desktop session packages.
- `hosts/zen/monitors.nix` keeps display layout out of the window manager modules.
- `modules/nixos/nix.nix` uses `nix-index-database`, disables channels, and keeps Nix fully flake-first.
- `modules/home/terminal/ai.nix` sources AI tools from `github:numtide/llm-agents.nix`.
